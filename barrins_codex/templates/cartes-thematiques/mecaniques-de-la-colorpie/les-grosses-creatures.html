{{ _("Les grosses créatures") }}

{% trans trimmed %} Une grosse créature, fatty, est une créature dont la force et l'endurance dépassent strictement 3. C'est l'opposée d'une créature weenie qui a peu de force et d'endurance. {% endtrans %}

{% trans trimmed %} Il est possible d'utiliser les grosses créatures pour deux objectifs différents : les créatures Stompy sont des grosses créatures très peu chères tandis que les Finisher permettent de clore la partie efficacement. {% endtrans %}

{% trans trimmed %} {{ tarmogoyf }} et {{ questing_beast }} sont deux créatures apparentées à la catégorie Stompy. Des créatures « anti-contrôle » se retrouvent aussi dans cette catégorie comme {{ thrun_the_last_troll }} et {{ shifting_ceratops }} jouées dans plusieurs formats. {{ strangleroot_geist }}, grâce à sa capacité Survivance, est l'icône des créatures Stompy en Modern. {% endtrans %}

{% trans trimmed %} À mi-chemin entre les créatures Stompy et les créatures Finisher, {{ elder_gargaroth }} bénéficie de mot-clés avantageux et d'une capacité très versatile qui permet à cette bête d'être jouée pour intervenir dans beaucoup de situations. {% endtrans %}

{% trans trimmed %} {{ primeval_titan }} se rapproche des créatures Finisher car la résolution de ce sort permet de développer de manière très rapide son mana et d'enchaîner les fatties pour prendre le contrôle du champ de bataille. {% endtrans %}

{% trans trimmed mana_dork=link("/cartes-thematiques/mecaniques-de-la-colorpie/les-mana-dorks", _("mana dorks")) %} Une partie des Finisher est jouée accompagnée de {{ mana_dork }} et de cartes comme {{ survival_of_the_fittest }}. L'intérêt est de jouer peu de Finisher et de pouvoir y avoir accès dès que c'est nécessaire. Ces Finisher sont souvent des créatures comme {{ craterhoof_behemoth }} et {{ kamahl_heart_of_krosa }} qui fournissent un effet {{ overrun }}. {% endtrans %}

{% trans trimmed reanimation=link("/cartes-thematiques/mecaniques-de-la-colorpie/la-reanimation-de-creatures", _("réanimation")) %} D'autres Finisher sont des cibles de choix pour les effets de {{ reanimation }}. {{ sidisi_brrod_tyrant }} peut permettre de développer un tel plan de jeu. Les créatures vertes à réanimer sont {{ archetype_of_endurance }}, {{ terastodon }} ou encore {{ woodfall_primus }}. {% endtrans %}

{% trans trimmed les_contresorts=link("/cartes-thematiques/mecaniques-de-la-colorpie/les-contresorts") %} C'est vraiment important que les fatties soient capables d'arriver sur le champ de bataille pour avoir l'occasion de faire ce pour quoi ces cartes ont été choisies. Pour ce faire, il faut réussir à résoudre son sort. Parmi les outils disponibles, il existe des cartes qui rendent des sort incontrables, c'est-à-dire que {{ les_contresorts }} ne peuvent pas faire ce pour quoi ils sont joués. {% endtrans %}
