{{ _("Classifier un deck en Duel Commander") }}

{% trans trimmed %} Au sens premier, le metagame est l'étude des différentes stratégies qu'un jeu permet. Avoir une vue globale des stratégies possibles et de leurs efficacités les unes par rapport aux autres est un point crucial pour progresser et améliorer son style de jeu. Vouloir regrouper les différents decks selon leurs points communs ou leurs startégies permet de mieux les étudier. Par conséquent, comment classer un deck en Duel Commander ? {% endtrans %}

{{ _("Deux méthodes s'opposent :") }}

  • {% trans trimmed %} L'approche en macrotype qui rassemble le maximum de decks dont le plan de jeu est similaire dans un minimum de groupes ; {% endtrans %}
  • {% trans trimmed %} L'approche par archétype qui rassemble des decks similaires en terme de cartes jouées. {% endtrans %}

{{ _("La Classification en macrotypes")}}

{% trans trimmed watching_the_clock=external("https://articles.starcitygames.com/premium/watching-the-clock/", "Watching the Clock") %} La classification est plébiscitée par Mike Mason dans son article {{ watching_the_clock }}. Elle se base sur la traditionnelle classification Aggro > Contrôle > Combo de Patrick Chapin et détaille le comportement des différentes stratégies qui existaient sous l'appelation Contrôle de P. Chapin. {% endtrans %}

{% trans trimmed %} Voici une présentation rapide des différents macrotypes développés par Mike Mason. Les différents archétypes trouvent tous une place dans les macrotypes suivants. {% endtrans %}

  • Aggro : {% trans trimmed %} Deck agressif dont le principal objectif est d'amener l'adversaire à zéro point de vie le plus rapidement possible en jouant des créatures ou des sorts de dégâts directs. {% endtrans %}
  • Tempo : {% trans trimmed %} Deck dont la stratégie consiste à poser rapidement une ou plusieurs menaces puis à les protéger suffisamment longtemps, le plus souvent à l'aide de contresorts, pour gagner. {% endtrans %}
  • {{ _("Contrôle") }} : {% trans trimmed %} Deck dont l'objectif est de ralentir le déroulement de la partie afin d'installer une guerre d'attrition basée essentiellement sur du Card Advantage. {% endtrans %}
  • Midrange : {% trans trimmed %} Deck ayant un plan de jeu proche de Contrôle mais qui va ensuite poser des créatures dont les forces et endurances sont supérieures aux créatures jouées par Aggro. {% endtrans %}
  • Combo : {% trans trimmed %} Deck basé sur une forte interaction entre deux cartes ou plus (une combinaison) qui permet de gagner la partie instantanément ou qui mène à un très fort avantage quasi- synonyme de victoire. {% endtrans %}

{{ _("La Classification en archetypes")}}

{% trans trimmed %} La classification en archétype permet d'affiner la classification en macrotypes en utilisant la tactique comme un élément de comparaison. {% endtrans %}

{{ _("Voici les principaux archétypes :") }}

  1. Swarm :

    {{ card_image("Thalia, Guardian of Thraben") }}

    {{ _("Catégorie de deck Aggro qui se caractérise par :") }}

    • {{ _("Gagne par l'accumulation de petites créatures ;") }}
    • {{ _("Gagne généralement le combat en débordant l'adversaire avec les créatures plutôt qu'en essayant de tuer les créatures adverses ;") }}
    • {{ _("Gagne en misant sur l'accumulation de créatures sans synergie particulière entre elles (en partageant un type par exemple).") }}

    {% trans trimmed %} Les principaux représentants de cette catégorie sont les decks Weenie White. Dans les formats éternels, comme le Modern ou le Legacy, ces créatures sont choisies pour gêner le développement adverse, voire le contrecarrer (ces créatures sont nommées hatebear). Ceci n'est toutefois pas une caractéristique générale des decks Swarm : la volonté principale reste d'envahir le champ de bataille (avec l'aide de {{ aether_vial }} en Modern et en Legacy). {% endtrans %}

    {% trans trimmed %} Si l'accumulation de créatures sur le champ de bataille est issue d'une combo ou si cette accumulation est susceptible de s'emballer à cause d'interactions ou de combos, le deck rentrera alors dans la catégorie Créature-Combo. Ces decks sont présentés plus tard et sont en général moins agressifs. {% endtrans %}

  2. Red Aggro :

    {{ card_image("Goblin Chainwhirler") }}

    {{ _("Catégorie de deck Aggro qui se caractérise par :") }}

    • {% trans trimmed %} Une courbe plus basse qui permet au deck de prendre l'ascendant sur l'adversaire dans les premiers tours ; {% endtrans %}
    • {% trans trimmed %} La possibilité de clore une partie avec des sorts de dégâts directs (des blasts) ; {% endtrans %}
    • {% trans trimmed %} Le besoin de se débarraser des bloqueurs adverses à l'aide des blasts se trouve en directe opposition avec la possiblité de cibler directement l'adversaire {% endtrans %}

    {% trans trimmed %} Bien que l'archétype s'appelle Red Aggro, la couleur rouge n'est absolument pas une nécessité pour faire partie de cette catégorie. Les decks Mono Green Stompy en sont d'ailleurs un bon exemple car les boosts, comme {{ giant_growth }}, ciblant des créatures avec capacités d'évasion ou des créatures bloquées remplacent le rôle des blasts. Les decks de cette catégorie font partie de ces decks plutôt simples à construire et relativement efficaces dans la plupart des formats. {% endtrans %}

    {% trans trimmed %} En fonction du ratio entre les sorts de dégâts directs et les sorts de créatures dans le deck, le nom du deck change : RDW / Sligh / Burn (voir la catégorie Lava Spike pour ce dernier). Souvent confondus, les decks RDW se distinguent pourtant notablement des decks Gobelins par leur absence ou quasi-absence de synergies "tribales". {% endtrans %}

  3. {{ _("Aggro Linéaire") }} :

    {{ card_image("Cavern of Souls") }}

    {{ _("Catégorie de deck Aggro qui se caractérise par :") }}

    • {% trans trimmed %} Les cartes sont exclusivement basées sur un thème commun, les créatures ont généralement le même type ; {% endtrans %}
    • {% trans trimmed %} Le deck utilise des synergies entre les cartes liées à ce thème commun. {% endtrans %}

    {% trans trimmed %} Beaucoup de decks Aggro Linéaire sont des decks tribaux, le thème commun étant un type de créature mais ce n'est toutefois pas une généralité. On retrouve aussi les decks Affinity et Tempered Steel dans cette catégorie dont le thème commun est ici « Artefact ». De plus, tous les decks tribaux ne sont pas des Aggro Linéaire, en effet Faeries est par exemple un deck Tempo et Elfball un deck Storm Combo. {% endtrans %}

    {% trans trimmed rebel_1999_block_deck=external("https://www.mtgtop8.com/event?e=9167&d=252441&f=BL", "deck bloc Rebel de 1999") %} Les deck Aggro Linéaire sont extrêmement populaires dans tous les formats et toutes les extensions récentes de Magic sont construites pour que cette catégorie de deck soit présente en limité. Le premier archétype à rentrer dans cette catégorie fut le {{ rebel_1999_block_deck }} {% endtrans %}

    {% trans trimmed rick_stout_uw_human=external("https://www.mtgtop8.com/event?e=2732&d=217663&f=ST","UW Human de Rick Stout") %} Il en faut parfois peu pour qu'un deck intègre cette catégorie. C'est par exemple le cas du deck standard {{ rick_stout_uw_human }} qui se distingua des autres listes UW par l'ajout de {{ cavern_of_souls }} et {{ champion_of_the_parish }} qui intégrait parfaitement à ce deck tribal. {% endtrans %}

  4. Suicide

    {{ card_image("Carnophage") }}

    {% trans trimmed %} Catégorie de deck Aggro (appelée Fish par Patrick Chapin) et qui se caractérise par : {% endtrans %}

    • {% trans trimmed %} Des accélérateurs de mana en vue de poser le plus vite possible des créatures pour mettre la pression sur l'adversaire ; {% endtrans %}
    • {% trans trimmed %} Des sorts de gestions qui empêchent l'adversaire de répondre aux menaces posées dans les premiers tours. {% endtrans %}

    {% trans trimmed %} L'appellation Suicide vient du fait que le deck à tendance à jouer des cartes qui vont l'affecter de manière néfaste. Certaines cartes peuvent nécessiter, par exemple, une perte de point de vie, un sacrifice de permanent ou encore de faire du Card Disadvantage, comme les accélérateurs de mana éphémères extrêmement courant dans ces decks. Un des premiers deck de cette catégorie fut le deck Extended Black Suicide de Brian Schneider (1999). {% endtrans %}

    {% trans trimmed %} On classe également dans cette catégorie le deck Dragon Stompy dans lequel on retrouve le besoin de mana dans les premiers tours via {{ simian_spirit_guide}}, {{ chrome_mox }} et {{ ancient_tomb }} pour mettre en place un léthal rapide et une carte qui va empêcher le développement adverse. Le deck mise sur cette impulsion qu'il aura mis sur le début de partie et pour laquelle il a dû faire énormément de CD. {% endtrans %}

  5. Tempo :

    {{ card_image("Delver of Secrets") }}

    {% trans trimmed %} Catégorie de deck Aggro-Contrôle, parfois assimilée aux decks Aggro et parfois considérée comme un macrotype à part entière. Les decks Tempo basent leur stratégie sur le fait de toujours vouloir conserver un meilleur tempo que celui de l'adversaire à l'aide de sorts peu cher. On parle aussi de Disruptive Aggro pour désigner cette famille de deck. {% endtrans %}

    {% trans trimmed %} Les decks Tempo sont connus pour jouer le rôle de l’agresseur contre Contrôle et le rôle du défenseur contre Aggro. {% endtrans %}

    {% trans trimmed %} Les principales faiblesses des decks Tempo sont qu'ils ne produisent pas de Card Advantage et que leur stratégie s'épuise sur le long terme. De plus, s'ils perdent leur avantage, il leur est difficile de reprendre l'ascendant sur la partie. Malgré cela, des decks reconnus comme étant les meilleurs decks de leur format appartiennent à cette catégorie. {% endtrans %}

  6. Rock :

    {{ card_image("Abrupt Decay") }}

    {{ _("Catégorie de deck Midrange qui se caractérise par :") }}

    • {{ _("Des créatures généralement un peu plus grosses qu'Aggro classique ;") }}
    • {{ _("De l'accélération de mana et de la disruption ;") }}
    • {{ _("Du vert pour les créatures et du noir pour la défausse et les removals.") }}

    {% trans trimmed %} Cette catégorie de deck vient d'un deck nommé par Malka un fan de catch : "The Rock and His Millions" où "The Rock" en hommage à Dwayne The Rock Johnson correspondait à la carte {{ phyrexian_plaguelord }} et "His Millions" correspondaient aux écureuils crées par {{ deranged_hermit }}. On peut distinguer principalement deux types de decks Rock : {% endtrans %}

    • {% trans trimmed %} Les decks plus orientés Contrôle qui se focalisent plutôt sur un Card Advantage progressif. Ces decks essentiellement B/G ont tendance à jouer le rôle du Contrôle même contre d'autres decks Contrôle ; {% endtrans %}
    • {% trans trimmed %} Les decks Rock plus Aggro qui se focalisent davantage sur la solidité de ses créatures que sur le Card Advantage. Ces decks correspondent essentiellement aux decks Junk (Abzan W/B/G). {% endtrans %}

  7. {{ _("Midrange Pur") }} :

    {{ card_image("Knight of the Reliquary") }}

    {{ _("Catégorie de deck Midrange qui se caractérise par :") }}

    • {{ _("La qualité intrinsèque des cartes plutôt que les synergies entre elles ;") }}
    • {{ _("Des créatures polyvalentes et des removals peu cher ;") }}
    • {{ _("La possibilité de s'adapter à n'importe quel environnement.") }}

    {% trans trimmed %} Les decks Pure Midrange sont parfois appelés Good Stuff car ils ont tendance à incorporer les meilleurs sorts du format, les removals les moins chers ainsi que les meilleures créatures pour chaque coût converti de mana. Cette configuration force bien souvent ces decks à jouer tricolore, voire quadricolore si la base de mana le permet. {% endtrans %}

    {% trans trimmed %} Les decks Pure Midrange sont des decks relativement faciles et rapides à construire une fois que sont identifiées les meilleurs cartes à disposition dans un format. Ils font donc très régulièrement partie des premiers decks à apparaître avec notamment les Aggro Linéaire et les Red Aggro. Ces decks ont aussi tendance à être mauvais contre les decks unfair et excellents contre les decks fair. C'est pourquoi dans un format résolu, lorsqu'un deck Midrange est capable d'incorporer au moins un sort unfair tel que de la destruction massive de terrain, il peut faire d'excellent résultats. {% endtrans %}

  8. {{ _("Midrange Non-Bleu") }} :

    {{ card_image("Gray Merchant of Asphodel") }}

    {{ _("Catégorie de deck Midrange qui se caractérise par :") }}

    • {{ _("De la gestion presque totalement dirigée contre les créatures adverses ;") }}
    • {% trans trimmed %} La présence d'un moteur de pioche (ou de Card Advantage) quelconque dont le deck est assez dépendant. {% endtrans %}

    {% trans trimmed mono_black_control=external("https://www.mtgtop8.com/event?e=9201&d=252718&f=BL","Mono Black Control"), astral_slide=external("https://www.mtgtop8.com/event?e=9207&d=252771&f=BL","Astral Slide") %} Bien que la majorité des decks de cette catégorie n'utilisent pas de mana bleu, ce n'est pas une condition nécessaire pour faire partie de cette catégorie. Les decks Non-Bleu sont globalement rares dans les formats eternels (voire absents du Legacy et du Vintage) car ils ne sont généralement pas assez polyvalents pour s'y imposer. Les decks Bloc {{ mono_black_control }} de 2002 et {{ astral_slide }} de 2003 font partie des permiers decks de cette catégorie. {% endtrans %}

    {% trans trimmed %} Le cas de {{ life_from_the_loam }} {% endtrans %}

    {% trans trimmed %} Cette catégorie permet d'aborder le cas de {{ life_from_the_loam }} présente dans des decks appartenant à des catégories diverses. L'un de ces decks, la version Aggro Loam de l'Extended que l'on retrouve aussi en Modern, vérifie justement les critères du deck Midrange Non-Bleu, {{ life_from_the_loam }} jouant ici le rôle du moteur de CA dont le deck est assez dépendant. {% endtrans %}

  9. Draw Go :

    {{ card_image("Cryptic Command") }}

    {{ _("Catégorie de deck Contrôle qui se caractérise par :") }}

    • {% trans trimmed %} Une grande quantité de contresorts pour interagir au maximum contre les actions adverses ; {% endtrans %}
    • {% trans trimmed %} Un moteur à Card Advantage pour compenser l'épuisement des contresorts ; {% endtrans %}
    • {% trans trimmed %} La présence de mass removals pour gérer l'ensemble des cartes qui ont pu se résoudre. {% endtrans %}

    {% trans trimmed %} Le terme Draw Go est une allusion aux tours rapides d'un joueur de cet archétype « My Turn, Draw, Go ! ». Ces decks se distinguent donc radicalement des decks Tapout vus précédemment. {% endtrans %}

    {% trans trimmed %} Les decks Draw Go ont tendance à faire du un pour un avec leurs contresorts. Ils sont naturellement désavantagés contre les decks Aggro qui jouent moins de terrains et donc piocheront plus de menaces que Draw Go ne pourra piocher de réponses. Pour contrebalancer cela, les decks Draw Go sont obligés de compenser avec du Card Advantage. {% endtrans %}

  10. Tapout :

    {{ card_image("Karn Liberated") }}

    {{ _("Catégorie de deck Contrôle qui se caractérise par :") }}

    • {% trans trimmed %} Des mass removal et des sorts visant à empêcher l'adversaire de jouer dans le but d'acheter du temps ; {% endtrans %}
    • {% trans trimmed %} Des sorts agressifs pour profiter au maximum du temps acheté. {% endtrans %}

    {% trans trimmed %} Les decks Tapout jouent essentiellement à leur propre tour et ne gardent donc généralement pas de manas disponibles lorsqu'ils ont fini de jouer. L'absence de contresort dans un deck Contrôle est généralement un bon moyen de repérer si un deck est un Tapout. {% endtrans %}

    {% trans trimmed %} La première vraie apparition de Tapout en tournoi date de 1997 avec le deck standard Big Blue de Erik Lauer. {% endtrans %}

  11. Prison :

    {{ card_image("Stasis") }}

    {{ _("Catégorie de deck Contrôle qui se caractérise par :") }}

    • {% trans trimmed %} La volonté de garder le champ de bataille sous contrôle plutôt que de lui opposer des menaces ; {% endtrans %}
    • {% trans trimmed %} Utilise ses sorts permissifs avant tout pour protéger sa propre stratégie ; {% endtrans %}
    • {% trans trimmed %} Une fin de partie où l'adversaire est privé d’interactions. {% endtrans %}

    {% trans trimmed %} Les decks Prison (aussi appelé Lock), comme les decks Tapout, ont tendance à utiliser leur mana de manière proactive. Toutefois, là où un deck Tapout va engager tout son mana pour poser une créature qui va dissuader son adversaire d'attaquer, un deck Prison préférera prévenir l'attaque, par exemple avec un {{ icy_manipulator }}. {% endtrans %}

  12. Contrôle-Combo Pur :

    {{ card_image("Crop Rotation") }}

    {{ _("Catégorie de deck Contrôle qui se caractérise par :") }}

    • {{ _("Un jeu entre Draw Go et Tapout avec une victoire issue d'une combo ;") }}
    • {% trans trimmed %} Une stratégie lente mais qui peut éventuellement menacer de gagner très tôt dans la partie ; {% endtrans %}
    • {{ _("De nombreux sorts de gestion et/ou de contresorts.") }}

    {% trans trimmed %} Contrairement à la majorité des decks Combo, les decks de cette famille peuvent faire du Card Advantage plutôt que de simplement creuser ou tutoriser les éléments de leur combo. On retrouve très souvent dans ces decks un questionnement sur la méthode à utiliser pour atteindre sa combo ou au contraire de contrôler le board si cela devient nécessaire. Voici quelques exemples : {% endtrans %}

    • {{ muddle_the_mixture }} → {{ thopter_foundry }} / {{ sword_of_the_meek }} (Combo Kill) ou {{ counterspell }}.
    • {{ bring_to_light }} → {{ scapeshift }} (Combo Kill) ou {{ damnation }}.
    • {{ crop_rotation }} → {{ dark_depths }} / {{ thespians_stage }} (Combo Kill) ou {{ glacial_chasm }}

  13. Ramp :

    {{ card_image("Primeval Titan") }}

    {{ _("Catégorie de deck Contrôle qui se caractérise par :") }}

    • {% trans trimmed %} La présence de nombreuses cartes productrices de mana et/ou de cartes permettant de mettre sur le champ de bataille plus d'un terrain par tour ; {% endtrans %}
    • {% trans trimmed %} La présence de cartes variées ayant un coût de mana élevé qui sont censées être jouées en mid game. {% endtrans %}

    {% trans trimmed %} Il ne faut pas confondre les decks Ramp avec les decks de la catégorie Big Spell bien que ceux-ci peuvent aussi avoir une optique Ramp en début de partie. {% endtrans %}

    {% trans trimmed %} Lorsqu'ils sont bien construits et qu'ils contiennent des sorts adaptés aux decks présents dans l'environnement de tournoi, les decks Ramp peuvent être redoutables. Ils sont faciles à piloter et ont une grande solidité sur le long terme. S'ils ne tuent pas dans le début de partie, ils ont toujours un plan B. {% endtrans %}

    {% trans trimmed %} Toutefois, si leur accélération de mana est sabotée, les decks Ramp sont terriblement ralentis dans leur stratégie et restent souvent au point mort. Ils ont globalement les mêmes problèmes que Midrange contre Contrôle et Tempo : s'ils sont pris de vitesse, ils doivent subir les sorts de gestion adverses. {% endtrans %}

  14. Big Spell :

    {{ card_image("Living End") }}

    {{ _("Catégorie de deck Combo qui se caractérise par :") }}

    • {% trans trimmed %} La présence d'un kill (le big spell) dont l'efficacité est due essentiellement au fait que tout le deck soit construit autour et pour maximiser l'effet de cette carte. {% endtrans %}
    • {% trans trimmed %} La résolution de ce big spell entraîne une victoire immédiate ou change radicalement la dynamique de la partie. {% endtrans %}

    {% trans trimmed %} Un aspect Ramp est généralement présent dans les decks Big Spell, mais contrairement aux decks Ramp, ces decks sont construits autour d'une carte en particulier. De plus l'aspect Ramp n'est absolument pas une caractéristique générale de cette catégorie, Living End est justement un exemple de deck Big Spell sans aucun aspect ramp. {% endtrans %}

    {% trans trimmed %} Le duo {{ scapeshift }} et {{ valakut }} {% endtrans %}

    {% trans trimmed %} Ce duo de cartes a donné son nom à de nombreux archétypes et tous ces archétypes ne correspondent pas aux même catégories. {% endtrans %}

    • {% trans trimmed %} La version Standard de Valakut est un deck Ramp qui se contente de jouer Valakut pour rentabiliser son aspect ramp sur le long terme, quitte à en faire son kill. Ici pas de Scapeshift. {% endtrans %}
    • {% trans trimmed %} Le deck Modern Scapeshift est un deck Contrôle-Combo Pur avec de nombreux sorts de gestions et contresorts pour temporiser et qui en fin de partie menace de tuer à tout moment en jouant Scapeshift. {% endtrans %}
    • {% trans trimmed %} Le deck Modern Valakut, quant à lui, est bien un deck Big Spell où tout le deck est construit autour d'un plan de jeu simpliste qui se résume à avoir le plus vite possible suffisamment de terrains en jeu et jouer son Big Spell : Scapeshift. {% endtrans %}
    • {% trans trimmed %} Enfin la version Omen Scapeshift se distingue foncièrement des autres par sa quasi-absence de montagne et donc à la nécessité d'avoir {{ prismatic_omen}} sur le champ de bataille avant de lancer scapeshift ce qui fait du deck un Combo Traditionnel. {% endtrans %}

  15. Combo Storm :

    {{ card_image("Grapeshot") }}

    {{ _("Catégorie de deck Combo qui se caractérise par :") }}

    • {{ _("L'accumulation des sorts et actions faites lors d'un même tour ;") }}
    • {{ _("L'absence de sorts spécifique, voire de séquence, pour gagner ;") }}
    • {{ _("Le besoin de quantité d'actions plus que de leur qualité.") }}

    {% trans trimmed %} Les decks Storm Combo ont un big turn à l'issue duquel l'adversaire est défait. Le nom de cette catégorie est tirée de la capacité Storm. Tous les decks avec une carte Storm en guise de kill appartiennent à cette catégorie. Ce ne sont pas les seuls, High Tide ou KCI font aussi partie de cette catégorie. {% endtrans %}

  16. {{ _("Combo Traditionnel") }} :

    {{ card_image("Painter's Servant") }}

    {{ _("Catégorie de deck Combo qui se caractérise par :")}}

    • {% trans trimmed %} Une condition de victoire basée directement ou indirectement sur une combo formée de deux ou trois cartes ; {% endtrans %}
    • {{ _("Des cartes permettant d'aller chercher les éléments de la combo.") }}

    {% trans trimmed %} Les stratégies des Combo Traditionnel peuvent être très variées. On peut en distinguer deux variantes : {% endtrans %}

    • {% trans trimmed %} Les decks qui vont avoir tendance à chercher dans leur bibliothèque les éléments de la combo ; {% endtrans %}
    • {% trans trimmed %} Les decks qui vont miser sur la redondance des éléments de la combo quitte à jouer parfois plusieurs combos différentes. {% endtrans %}

    {% trans trimmed %} Dans cette seconde variante, on peut citer les decks Reanimator ou, en Legacy, les decks {{ show_and_tell }} qui incorporent aussi {{ sneak_attack }}. {% endtrans %}

    {{ _("Le plan de jeu Reanimator") }}

    {% trans trimmed %} Une fois réanimé, {{ griselbrand }} à tendance à transformer le plan de jeu Reanimator en quelque chose proche d'un Storm Combo mais le plan de jeu originel du deck correspond bien à la catégorie Combo Traditionnel. {% endtrans %}

  17. Lava Spike :

    {{ card_image("Lava Spike") }}

    {{ _("Catégorie de deck Aggro-Combo qui se caractérise par :") }}

    • {{ _("Une absence d'interactivité avec l'adversaire ;") }}
    • {% trans trimmed %} Une stratégie qui consiste à cibler avec ses sorts directement l'adversaire. {% endtrans %}

    {% trans trimmed %} On retrouve dans cette catégorie principalement deux sortes de deck : Burn et Meule : {% endtrans %}

    • {% trans trimmed %} Les decks Burn utilisent leurs sorts pour infliger directement des dégâts à l'adversaire. Ils sont présents dans tous les formats. Ils sont tous à dominante rouge, mais tous ne sont pas nécessairement monocolores ; {% endtrans %}
    • {% trans trimmed %} Les decks Meule utilisent leurs sorts pour mettre les cartes de la bibliothèque de l'adversaire dans son cimetière. Ils sont absent des formats éternels et correspondent pour la plupart au plan de jeu de certains decks limités bleus. {% endtrans %}

  18. Wave Combo :

    {{ card_image("Bloodghast") }}

    {{ _("Catégorie de deck Aggro-Combo qui se caractérise par :") }}

    • {% trans trimmed %} Une construction autour d'une mécanique avec des cartes très synergiques entre elles ; {% endtrans %}
    • {% trans trimmed %} Une stratégie qui peut menacer extrêmement rapidement l'adversaire et se répéter par vagues successives sur le long terme ; {% endtrans %}
    • {{ _("Une absence de réel Big Turn.") }}

    {% trans trimmed %} Les principaux decks Wave Combo sont construits autour d'une mécanique dont ils abusent des effets. Les deux decks principaux de cette catégorie sont Dredge qui abuse de la capacité éponyme et des synergies avec le cimetière et {{ hollow_one }} basé quant à lui sur les effets de défausse. {% endtrans %}

    {% trans trimmed %} Cette catégorie a finalement connu très peu de représentants et il est compliqué d'en lister des caractéristiques arrêtées. Toutefois, les deux decks cités ont la particularité d'être relativement imprévisibles dans leur sortie même pour leur pilote. {% endtrans %}

  19. Voltron :

    {{ card_image("Glistener Elf") }}

    {{ _("Catégorie de deck Aggro-Combo qui se caractérise par :") }}

    • {% trans trimmed %} Une condition de victoire basée sur le fait d'attaquer avec des créatures (généralement une seule) pour tuer en peu de phases d'attaque (parfois une seule) ; {% endtrans %}
    • {{ _("Un rythme de jeu plus rapide que les decks Aggro classiques.") }}

    {% trans trimmed %} Les decks Voltron empruntent certaines caractéristiques aux catégories Combo Traditionnel et Storm Combo. Un deck emblématique de cette catégorie est {{ kiln_fiend }} : {% endtrans %}

    • {% trans trimmed %} Il s'agit d'une combo deux cartes avec {{ berserk }} ou {{ temur_battle_rage }}, ce qui donne le côté Combo Traditionnel ; {% endtrans %}
    • {% trans trimmed %} Toutefois, le deck n'a pas besoin de cette seconde carte pour gagner et peut remporter la victoire simplement par l'accumulation de sorts joué lors d'un même tour, ce qui donne l'aspect Storm Combo. {% endtrans %}

    {% trans trimmed %} On retrouve aussi dans cette catégorie les decks hexproof qui se basent sur une créature qui grossit jusqu'à devenir létale. {% endtrans %}

    {{ _("L'origine de Voltron") }}
